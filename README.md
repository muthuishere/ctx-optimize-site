# ctx-optimize — site

GitHub Pages site for [ctx-optimize](https://github.com/muthuishere/ctx-optimize): a deterministic code knowledge graph for coding agents. One static Go binary, no LLM at query time, no database, no server.

- Live: https://muthuishere.github.io/ctx-optimize-site/
- Benchmarks: raw data in `bench/results.json`, methodology on the page.
- Demos: recorded with [vhs](https://github.com/charmbracelet/vhs), unedited.
